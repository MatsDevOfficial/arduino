# arduino

## NLT
Pillendoos voor ouderen met dementie, sluit de arduino aan op Raspberry Pi5 GPIO 17, 27 en GND
| Richting          | Arduino | Raspberry Pi |
| ----------------- | ------- | ------------ |
| Start             | Pin 2   | GPIO 17      |
| Stop              | Pin 4   | GPIO 27      |
| Got it knop       | Pin 3   | - (alleen naar arduino) |
| GND               | GND     | GND          |
